# Haukcode.StreamDeck

Transport-agnostic .NET library for Elgato Stream Deck devices (USB HID + Network Dock).

Single `IStreamDeckDevice` interface works whether the deck is plugged into USB or connected behind an Elgato Network Dock. Includes mDNS discovery (`_elg._tcp`) and USB enumeration.

**Supports:** MK.2 · XL · Plus (encoders) · Studio · Network Dock TCP protocol

See [GitHub](https://github.com/HakanL/Haukcode.StreamDeck) for full documentation and samples.
